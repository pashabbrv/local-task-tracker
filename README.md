## Local Task Tracker
Локальное десктоп‑приложение на Electron для трекинга личных задач по проектам. Все данные хранятся только на компьютере пользователя.

### Требования
- **OS**: Windows 10/11
- **Node.js**: рекомендуемая версия 18+
- **npm** (обычно идёт вместе с Node.js)

### Установка и запуск из исходников (для разработчика)

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/pashabbrv/local-task-tracker.git
   cd local-task-tracker
   ```

2. Установить зависимости:
   ```bash
   npm install
   ```
   Если не установился `electron` или `electron-builder`, можно поставить их явно:
   ```bash
   npm install --save-dev electron electron-builder
   ```

3. Запустить приложение в режиме разработки:
   ```bash
   npm start
   ```

### Сборка установщика для Windows

1. В корне проекта выполнить:
   ```bash
   npm run dist
   ```

2. После сборки в папке `dist` появится установочный файл вида:
   ```text
   Local Task Tracker Setup X.Y.Z.exe
   ```

3. Этот `.exe` можно передать любому пользователю Windows — он установит приложение как обычную программу.

### Как установить приложение конечному пользователю

1. Скачать/получить файл установщика `Local Task Tracker Setup X.Y.Z.exe` (из папки `dist`).
2. Дважды кликнуть по файлу и пройти стандартный мастер установки (можно выбрать путь установки).
3. После установки приложение появится в системе (ярлык в меню Пуск/на рабочем столе — зависит от настроек мастера).
4. Все данные (проекты и задачи) хранятся локально в профиле пользователя Windows (без регистрации и авторизации).
